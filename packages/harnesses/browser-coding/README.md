# @ecp/harnesses-browser-coding

Browser Coding harness (`@ecp/harness-browser-coding`) for local coding models (Ollama `qwen2.5-coder:1.5b` eval matrix).

- **TypeScript-first:** Fluent workflows and typed `EcpIntent` / `HarnessReply` modules (no EQL).
- **Eval matrix only** — browser demo continues to use `@ecp/harnesses-browser-nano`.

```ts
import { registerBrowserCodingHarnesses, BROWSER_CODING_HARNESS_CAPABILITY } from "@ecp/harnesses-browser-coding"
```
